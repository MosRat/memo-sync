Bundled UI fonts
================

These WOFF2 files are packaged with the desktop/web frontend so Memo Sync uses
consistent UI and code typography even when Inter or JetBrains Mono are not
installed on the host system.

- `InterVariable.woff2` provides the UI font.
- `JetBrainsMono-VF.woff2` provides the editor and code font.

Both fonts are distributed under the SIL Open Font License 1.1.
