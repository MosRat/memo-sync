Embedded preview fonts
======================

Memo Sync embeds these fonts so Typst previews render the same way on Windows,
macOS, Linux, and small server images without depending on system font
installation:

- `NotoSerifSC-VF.ttf` for Chinese and mixed-language literary previews.
- `NotoSansSC-VF.ttf` for compact and technical previews.
- `InterVariable.ttf` for bundled Latin UI-style text in compact and technical previews.
- `JetBrainsMono-VF.ttf` for bundled code typography in Typst previews.
- `LXGWWenKai-Regular.ttf` for the notebook-style Chinese preview template.
- `CascadiaCode.ttf` remains as a secondary code font fallback.

Noto fonts, Inter, JetBrains Mono, LXGW WenKai, and Cascadia Code are
distributed under the SIL Open Font License 1.1.
