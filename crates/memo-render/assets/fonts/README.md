Embedded preview fonts
======================

Memo Sync embeds these fonts so Typst previews render the same way on Windows,
macOS, Linux, and small server images without depending on system font
installation:

- `NotoSerifSC-VF.ttf` for Chinese and mixed-language literary previews.
- `NotoSansSC-VF.ttf` for compact and technical previews.
- `CascadiaCode.ttf` for code blocks.

Noto fonts are distributed under the SIL Open Font License 1.1. Cascadia Code is
distributed under the SIL Open Font License 1.1.
